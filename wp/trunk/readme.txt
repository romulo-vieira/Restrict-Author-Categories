Restrict on which categories a user can post. Just that!

== Description ==

Restrict on which categories a user can post.

It also prevents the users from creating/editing/deleting categories.

Only Admins can set other user's categories.

Works with Block Editor and Classic Editor.

== Frequently Asked Questions ==

= Does this plugin works on Block/Gutenberg Editor? =

Sure it does.

= Can I restrict categories of any User? =

All users, except Admins.

= Does it changes WP Roles =

No. This plugins does not affect or change any of the WP Roles.
